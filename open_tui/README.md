### openTUI 
- framework used for temrinal.shop & [openCode CLI](https://opencode.ai/docs/custom-tools/)
- TS 

## to run
- have bun installed 

```bun install @opentui/core```

- (from the repo root - i.e. open_tui/codestation/src)
- install required deps  

```bun install``` 

- run example TUI  

```bun run index.ts```
